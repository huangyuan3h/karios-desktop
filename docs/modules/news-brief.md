# News Brief 模块

> 财经新闻聚合与摘要模块

---

## 模块定位

News Brief 模块负责从多个财经新闻源聚合新闻，并使用 AI 生成摘要，帮助用户快速了解市场动态和重要事件。它是**信息获取**的辅助工具。

## 核心价值

- **新闻聚合**：从多个来源聚合财经新闻
- **AI 摘要**：自动生成新闻要点摘要
- **重要标记**：标记重要新闻方便回顾
- **时间过滤**：按时间范围筛选新闻

---

## 核心概念

### 新闻源（Source）

系统支持添加多个 RSS 新闻源：

- 默认源：内置常用财经新闻源
- 自定义源：用户可添加自己的 RSS 源

### 新闻项（News Item）

每条新闻包含：

| 字段 | 说明 |
|------|------|
| Title | 新闻标题 |
| Link | 原文链接 |
| Summary | 摘要内容 |
| Published At | 发布时间 |
| Fetched At | 抓取时间 |
| Is Important | 是否标记重要 |

### AI 摘要

系统会定期使用 AI 对最近 24 小时的新闻生成摘要：

- 提取关键信息
- 过滤无关内容
- 聚焦财经相关话题

## 业务流程

### 日常使用流程

```
1. 新闻抓取
   ├── 点击 "Fetch" 从 RSS 源抓取最新新闻
   └── 或在 Dashboard 点击 "Sync all"

2. 浏览新闻
   ├── 查看最新新闻列表
   ├── 点击新闻标题打开原文
   └── 查看摘要内容

3. 标记重要
   ├── 点击星标标记重要新闻
   └── 重要新闻单独显示在顶部

4. AI 摘要
   ├── 在 Dashboard 查看 AI 生成的新闻摘要
   └── 了解市场整体动态

5. 深入分析
   ├── 将新闻添加到 AI 对话
   └── 结合其他数据进行深入分析
```

### 新闻管理

```
设置页面
     │
     ├── 启用/禁用新闻源
     ├── 添加自定义新闻源
     ├── 编辑新闻源名称
     └── 删除不需要的新闻源
```

## 与其他模块的关系

```
                    ┌──────────────────┐
                    │   News Brief     │
                    └────────┬─────────┘
                             │
                   ┌─────────┴─────────┐
                   │                   │
                   ▼                   ▼
            ┌────────────┐      ┌────────────┐
            │  Dashboard │      │    AI      │
            │ (摘要展示) │      │ (新闻分析) │
            └────────────┘      └────────────┘
```

### 与 Dashboard 的关系

- Dashboard 展示 AI 生成的新闻摘要
- 用户无需进入 News 页面即可了解市场动态
- 摘要每 4 小时更新一次

### 与 AI 的关系

- 新闻数据可作为 AI 分析上下文
- AI 可以解释新闻对市场的影响
- AI 可以结合新闻和其他数据给出建议

## 典型使用场景

### 场景一：每日信息获取

1. 打开 Dashboard
2. 查看新闻摘要区域
3. 快速了解市场动态
4. 发现重要新闻后点击查看原文

### 场景二：重要新闻追踪

1. 浏览新闻列表
2. 发现与持仓相关的新闻
3. 点击星标标记重要
4. 后续在 Starred 区域快速回顾

### 场景三：深度分析

1. 将重要新闻作为上下文添加到 AI 对话
2. 让 AI 分析新闻对持仓的影响
3. 结合技术面给出操作建议

---

## 关键业务规则

### 新闻源管理

- 可以添加任意 RSS 源
- 建议选择高质量财经新闻源
- 避免信息过载

### 摘要更新频率

- AI 摘要每 4 小时更新一次
- 强制同步时会重新生成
- 摘要缓存在本地

### 时间过滤

支持的时间范围：
- Last 6h
- Last 12h
- Last 24h（默认）
- Last 48h
- Last 72h

---

## 数据来源

### 默认新闻源

系统内置的默认新闻源，可在 Settings 添加。

### 自定义新闻源

用户可以添加自己的 RSS 源：
1. 点击 "Add Custom"
2. 输入名称和 RSS URL
3. 启用后即可抓取

---

## 注意事项

### 信息准确性

- 新闻内容来自第三方源
- 不保证信息的准确性和时效性
- 重要决策请核实原文

### AI 摘要局限

- AI 摘要可能遗漏重要信息
- 摘要仅供参考
- 建议查看原文获取完整信息

### 信息过载

- 不要订阅过多新闻源
- 使用星标筛选重要新闻
- 定期清理不重要的标记
