# Industry Flow 模块

> 行业资金流向分析模块

---

## 模块定位

Industry Flow 模块追踪 A 股市场各行业的资金流入流出情况，帮助用户发现资金集中流入的热门板块和资金撤离的冷门板块，是**板块轮动分析**的核心工具。

## 核心价值

- **资金流向追踪**：展示各行业的净流入/流出金额
- **热点发现**：识别资金集中流入的板块
- **趋势判断**：通过多日数据判断资金流向趋势
- **主线识别**：通过评分系统识别当前市场主线

---

## 核心概念

### 行业资金净流入

行业资金净流入 = 该行业所有股票的净流入之和

- **正值**：资金净流入，表示资金正在流入该行业
- **负值**：资金净流出，表示资金正在流出该行业

### 时间窗口

系统提供多个时间窗口的资金流向数据：

| 时间窗口 | 说明 | 用途 |
|----------|------|------|
| 1D（当日） | 今日资金净流入 | 发现当日热点 |
| 5D（5日累计） | 最近5日累计净流入 | 判断短期趋势 |
| 10D（10日累计） | 最近10日累计净流入 | 判断中期趋势 |

### Mainline（主线）评分

主线评分 = Flow Score + Breadth Score + Trend Score

系统会自动计算每个行业的主线评分，用于识别当前市场主线：

#### Flow Score（资金流向分数）

| 指标 | 权重 |
|------|------|
| 20日累计净流入 | 基础分 |
| 5日累计净流入 | 加分项 |
| 20日排名 | 加分项 |
| 5日排名 | 加分项 |
| 10日内正流入天数 | 加分项 |
| 中期积累信号 | 加分项 |
| 短期强度信号 | 加分项 |
| 一致性信号 | 加分项 |

#### Breadth Score（广度分数）

| 指标 | 说明 |
|------|------|
| 涨停数 | 该行业涨停股票数量 |
| 涨停排名 | 在所有行业中的排名 |
| 涨停是否达标 | 数量是否达到阈值 |
| 龙头数 | 龙头股数量 |
| 龙头是否达标 | 数量是否达到阈值 |
| 暴涨比例 | 大涨股票占比 |

#### Trend Score（趋势分数）

| 指标 | 说明 |
|------|------|
| 指数在 MA20 上方 | 行业指数趋势 |
| MA20 向上 | 均线方向 |
| RPS | 相对强度指标 |
| RPS 是否达标 | 强度是否足够 |

#### 主线认定条件

一个行业被认定为"当前主线"需要满足：

1. 总分 > 80 分
2. 连续 3 个交易日以上保持高分

## 业务流程

### 日常使用流程

```
1. 数据同步
   ├── 盘后点击 "Sync latest" 获取当日数据
   └── 或使用 "Force sync" 强制重新获取

2. 热点发现
   ├── 查看 "Daily top inflow (Top5 × Date)" 表格
   ├── 查看热点行业工作流卡片
   └── 识别资金集中流入的板块

3. 主线确认
   ├── 查看 "Mainline (主线)" 区域
   ├── 确认当前市场主线行业
   └── 结合评分细节分析

4. 决策支持
   ├── 将数据复制为 Markdown
   ├── 添加到 AI 对话进行分析
   └── 结合其他模块做决策
```

### 热点行业工作流

系统会自动识别"热点行业"，遵循以下规则：

**优先级一：动量突破信号**

- 今日净流入 > 20亿
- 排名提升 > 10名（相比昨天）
- 这通常是新主线的第一天，爆发力更强

**优先级二：日内 Top 与 5 日 Top 交集**

- 今日净流入排名靠前
- 5 日累计净流入排名也靠前
- 持续获得资金关注的行业

**操作建议**：只从这 3 个热点行业中选择符合技术条件的股票加入 Watchlist。

### 与 Watchlist 的协作

当股票所在行业处于热点 Top3 时，该股票的 Score 会获得额外加分：

- 这反映了"选股先选板块"的投资理念
- 处于热点行业的股票更容易获得资金推动

### 与 Screener 的协作

- 通过 Screener 筛选股票时，可以参考 Industry Flow 数据
- 优先选择处于热点行业的股票
- 避免选择资金持续流出行业的股票

## 典型使用场景

### 场景一：每日盘后复盘

1. 盘后打开 Industry Flow 页面
2. 点击 "Sync latest" 同步当日数据
3. 查看当日 Top5 流入行业
4. 查看热点行业工作流卡片
5. 识别今天哪些板块获得了资金关注

### 场景二：确认主线方向

1. 查看 Mainline 区域
2. 确认当前市场主线行业
3. 分析各评分维度（Flow/Breadth/Trend）
4. 判断主线是否持续或即将切换

### 场景三：板块轮动预警

1. 对比 1D 和 5D 的流入排名
2. 发现排名大幅提升的行业（动量突破）
3. 发现排名大幅下降的行业（资金撤离）
4. 及时调整持仓方向

## 关键业务规则

### 资金流向计算规则

```
行业净流入 = Σ (个股净流入)

个股净流入计算：
- 大单净流入 = 大单买入金额 - 大单卖出金额
- 中单净流入 = 中单买入金额 - 中单卖出金额
- 小单净流入 = 小单买入金额 - 小单卖出金额

综合净流入 = 大单 × 权重1 + 中单 × 权重2 + 小单 × 权重3
```

### 排名变化信号

```
排名变化 = 昨天排名 - 今天排名

正值：排名提升（从后面排到前面）
负值：排名下降

动量突破条件：
- 今日净流入 > 20亿
- 排名提升 > 10名
```

## 数据来源

数据来自 A 股市场：

- 使用 akshare 等数据源获取行业资金流向
- 每日收盘后更新
- 数据存储在本地数据库

---

## 注意事项

### 数据时效性

- 建议盘后同步，确保数据完整
- 盘中数据可能不完整或不准确

### 行业分类

- 使用申万一级行业分类
- 共 31 个行业

### 主线切换

- 主线不会频繁切换
- 一般持续数周甚至数月
- 关注评分趋势而非单日波动
